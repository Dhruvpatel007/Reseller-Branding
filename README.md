#Branding App with MERN stack


## Available Scripts

STEP 1. Install NPM packages:

### `cd ./backend`
### `npm install`
### `cd..`
### `cd ./frontend`
### `npm install`

STEP 2. Set up environment variable:

PORT=
MONGO_URI=
JWT_SECRET=


STEP 3. Import data to database
### `cd ./backend`
### `node seeder.js`

STEP 4. Run the application:
### `cd ./backend`
### `npm start`
### `cd..`
### `cd ./frontend`
### `npm start`
